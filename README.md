# PowerBi_Dashboard_project
📌 Conclusion / Insights

This dashboard helps in understanding overall sales performance, customer buying behavior, and product trends. Through the analysis, it was observed that discounts significantly influence sales quantity, while some categories consistently generate higher revenue than others. Seasonal sales trends and top-performing products were identified to support better business decisions.

The dashboard provides interactive and data-driven insights that can help businesses optimize pricing strategies, improve product performance, and track growth effectively over time.
